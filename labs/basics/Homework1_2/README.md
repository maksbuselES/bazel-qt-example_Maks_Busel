# Homework 2 — Редактор карточек людей

## Идея приложения

Это приложение для создания и редактирования карточек людей в стиле коллекционной карточной игры.

У каждой карточки есть:
- фото
- имя
- возраст
- профессия
- статус
- редкость
- описание

Карточки отображаются в списке слева как мини-карточки с аватаром и цветовой рамкой редкости.

## Используемые виджеты

### Basic widgets
1. **QLineEdit**
   - имя
   - профессия
   - статус

2. **QSpinBox**
   - возраст

3. **QComboBox**
   - редкость

4. **QPushButton**
   - новая карточка
   - удалить карточку
   - загрузить фото
   - сохранить
   - перезагрузить

### Другие виджеты
5. **QListWidget**
   - список карточек слева

6. **QTextEdit**
   - описание персонажа

Дополнительно используются:
- `QLabel` для отображения фото и имени
- `QFileDialog` для выбора изображения
- `QMessageBox` для сообщений
- `QGroupBox` для визуального разделения интерфейса

## Как пользоваться

### Список слева
- Выберите карточку кликом мыши.
- Кнопка **Новая карточка** создаёт нового персонажа.
- Кнопка **Удалить карточку** удаляет текущую карточку.

### Редактор справа
После выбора карточки можно редактировать:
- имя
- возраст
- профессию
- статус
- редкость
- описание

### Фото
Нажмите **Загрузить фото**, чтобы выбрать изображение для текущей карточки.

### Сохранение
- **Сохранить** — записывает текущее состояние карточек в файл `cards.json`
- **Перезагрузить** — перечитывает карточки из файла

## Где лежат данные

Карточки хранятся в файле:

`labs/basics/Homework2/data/cards.json`

Можно:
- редактировать карточки через приложение
- либо править JSON вручную

## Редкости

Поддерживаются следующие редкости:
- Нормальная
- Редкая
- Эпическая
- Мифическая
- Легендарная
- ???

У каждой редкости:
- свой цвет рамки фото
- свой градиент
- свой фон карточки и редактора

## Особенности

- Возраст изменяется только вводом с клавиатуры
- Статус — это свободное текстовое поле
- Интерфейс адаптируется к изменению размера окна
- Используются layout'ы
