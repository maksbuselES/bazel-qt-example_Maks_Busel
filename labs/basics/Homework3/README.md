# Лабораторная работа №3: Language Learning App

Интерактивное приложение для изучения иностранного языка на C++/Qt6, разработанное в рамках учебного курса. Приложение вдохновлено стилистикой DuoLingo и объединяет динамический интерфейс с системой игрофикации.

## 📋 Выполнение требований ТЗ (10/10)

1.  **Layout**: Все виджеты организованы с помощью `QVBoxLayout` и `QHBoxLayout`. Интерфейс адаптивен и корректно масштабируется при изменении размеров окна.
2.  **Dynamic widgets**: Реализована динамическая смена контента через `QStackedWidget`. Переходы между меню и типами упражнений происходят в одном окне без создания лишних сущностей.
3.  **Menu**: Присутствует `MenuBar` с возможностью изменить уровень сложности через **отдельное диалоговое окно** (`QDialog`), как указано в требованиях.
4.  **Translation Mode**: Реализован режим перевода (PushButton + TextEdit). Пользователь вводит ответ с клавиатуры, после чего кнопка "Submit" производит оценку и переход.
5.  **Grammar Mode**: Реализовано упражнение на грамматику (PushButton + RadioBox). Пользователь выбирает один верный вариант из нескольких предложенных.
6.  **Progress bar**: Визуализирует количество выполненных заданий в рамках текущего набора из $N$ упражнений.
7.  **Grading system**: 
    *   Введена система "жизней" (Mistakes $M$).
    *   Баллы начисляются только в случае успешного выполнения всего набора заданий.
    *   При превышении лимита ошибок сессия завершается.
8.  **Timer**: При старте упражнения запускается таймер. По истечении времени выдается сообщение, и упражнение прерывается.
9.  **Help (Клавиша "H")**: Реализована обработка нажатия клавиши "H", которая выводит информационное окно с подсказкой к текущему заданию.
10. **Design**: Приложение имеет презентабельный вид в стиле DuoLingo (QSS стили, Toast-уведомления, кастомные кнопки).

## 🚀 Продвинутые функции (Extra Points)

*   **Алгоритм Левенштейна**: Реализован продвинутый алгоритм сравнения строк для режима перевода. В настройках можно включить "Режим опечаток", позволяющий простить 1 ошибку в слове.
*   **Система пропусков (Skip)**: Реализована возможность пропустить текущее задание ценой одной "жизни" (сердечка).
*   **Рандомизация**: Задания выбираются случайным образом из обширного репозитория (80+ задач), разделенного на уровни Easy, Medium и Hard.
*   **Toast Notifications**: Всплывающие уведомления "ВЕРНО" и "ОШИБКА", которые автоматически исчезают через 1 секунду.

## 🛠 Сборка и запуск

Проект собирается с помощью системы **Bazel**.

1.  Перейдите в корень проекта:
    ```bash
    cd ~/bazel-qt-example
    ```
2.  Выполните полную очистку и запуск:
    ```bash
    bazel clean --expunge && bazel run //labs/basics/Homework3:lang_app
    ```

## ⌨️ Управление
*   **Enter / Return**: Подтвердить ввод / Начать игру.
*   **Клавиша H**: Вызвать подсказку.
*   **Кнопка Skip**: Пропустить задание (-1 жизнь).

## 📁 Структура файлов проекта
*   `src/main_window.h / .cpp` — Основной интерфейс и логика переключений.
*   `src/session_engine.h / .cpp` — Логика таймера, жизней и начисления очков.
*   `src/exercises.h` — Реализация типов заданий (Translation и Grammar).
*   `src/task_repository.h` — База данных упражнений с разделением по сложностям.
*   `src/fuzzy_matcher.h` — Алгоритм расстояния Левенштейна.
*   `src/style.h` — Таблицы стилей (QSS) для дизайна.
*   `src/main.cpp` — Точка входа в приложение.
